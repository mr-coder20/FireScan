# 🤝 Contributing to FireScan

First off, thank you for considering contributing! 🎉

## Code of Conduct

Be respectful, inclusive, and professional. No harassment will be tolerated.

## How to Contribute

### Report a Bug
1. Check if it's already reported in [Issues](https://github.com/mr-coder20/FireScan/issues)
2. Use the bug report template
3. Include: OS, Go version, steps to reproduce, expected vs actual behavior

### Suggest a Feature
1. Open a [Feature Request](https://github.com/mr-coder20/FireScan/issues/new)
2. Explain the use case clearly
3. Include examples if possible

### Submit Code

```bash
# Fork the repo
git clone https://github.com/YOUR-USERNAME/FireScan.git
cd FireScan

# Create a branch
git checkout -b feat/your-feature-name

# Make changes
# ...

# Run tests
make test

# Commit (Conventional Commits)
git commit -m "feat(scanner): add new engine type"

# Push
git push origin feat/your-feature-name

# Open a Pull Request
